# Milin Tetris-peli

Harjoitustyön aiheena on yksinkertaisella logiikalla toimiva tetris-peli, jossa logiikka koostuu putoavista palikoista, pisteenlaskusta, tauotuksesta ja nopeutuvasta putoamisesta (tasot).
